# Countdown timer

app to countdown given amount of seconds in bash

## install

## use

```
$ node app.js timer 123
```

## set link

```
$ sudo npm link timer
```

now Yout can use it in bash like:

```
$ timer s 2700
```

# delete link

```
$ sudo npm unlink timer
```

## when git pull

```
$ git checkout app.js
```
